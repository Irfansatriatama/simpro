Import / Export

Backup data, restore, dan migrasi dari Trello atau Jira.

Export
Export SIMPRO JSON
Backup semua data (project, task, user, sprint, dll) ke satu file JSON.
Export Task ke CSV
Download daftar task satu project dalam format CSV (Key, Title, Status, Assignee, dll).
Import
Import SIMPRO JSON
Restore backup atau gabungkan data dari file SIMPRO JSON.
Semua data yang ada akan dihapus permanen. Ketik CONFIRM untuk lanjut.
Import dari Trello
Import board Trello ke project SIMPRO melalui wizard 3 langkah.
Cara export dari Trello: Buka board → Menu (···) → Print, Export, and Share → Export as JSON.
Import dari Jira CSV
Import issue Jira dari file CSV dengan mapping kolom yang bisa disesuaikan.
Cara export dari Jira: Buka project → Issues → filter yang diinginkan → Export → Export CSV (current fields).
